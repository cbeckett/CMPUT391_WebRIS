Installation

Getting Started ››

  1. Unpack the RIS package onto a local directory
  2. In dbScripts, run setup.sql in a Maria DBMS (ex: HeidiSQL) to set-up the required database.
  3. Ensure that the root password for MariaDB is set to 12345 and that the port is port 3306
  4. Copy the CMPUT_391 folder over to the Tomcat server's webapps folder
  5. Restart the Tomcat server
  6. Navigate to http://ui<COMPUTER_NUMBER>.cs.ualberta.ca:<PORT_NUMBER>/CMPUT391_RIS/
