Uploading

Modules ››

Users can navigate to this module by selecting the "Upload" button in the navigation bar.



If the current user is a Radiologist class, access to this page will be granted.  Else, the user will be redirected to the Access Denied page where which they can still use the navigation bar to access other modules.



To upload an image, click the browse button to select the chosen image.


After the image is selected, click upload and you will be notified if the upload was successful or not.

Click the back button in your browser to navigate back to the RIS system.
