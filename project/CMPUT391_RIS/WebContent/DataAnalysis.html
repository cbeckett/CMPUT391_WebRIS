Users can navigate to this module by selecting the "Data Analysis" button in the navigation bar.


If the current user is an Administrator class, access to this page will be granted.  Else, the user will be redirected to the Access Denied page where which they can still use the navigation bar to access other modules.


To generate the OLAP report, the user has the option to enter 6 criteria; Patient First Name, Patient Last Name, Test Type, Test Year, Test Month, and Time period.  The Time period input allows the user to specify how the OLAP data will be grouped.  By default the data is lumped together into one dataset but the user has the option to lump the OLAP data by year, month, and week of year.  The user can enter any combination of these 6 criteria (or none).  When the user clicks submit, the OLAP report data will be generated and displayed in a table underneath the OLAP data form.
