Search

Modules ››

Users can navigate to this module by selecting the "Search" button in the navigation bar.


This module can be accessed by all user classes and thus is the default module upon entering the system.


The user can enter their desired search criteria.  When the click search, a list of thumbnails are available for them to browse.


The user can click the return button to navigate back to the search module.


The user can also use the "Picture Size" column to view the image in regular size and big size.
